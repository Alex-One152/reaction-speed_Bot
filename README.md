# reaction-speed_Bot
Бот для теста на скорость реакции на сайте https://mozgion.ru/test-trenazher-na-skorost-reakcii/
Для работы бота потребуется установить модули PyAutoGUI, mss и numpy

pip install PyAutoGUI
pip install mss
pip install numpy
